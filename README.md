# InvenData Lite 📦
**Sistema de Gestión de Inventarios para el Proyecto Integrador**

InvenData Lite es una solución ágil diseñada para el control de existencias en PyMEs, El enfoque principal es la simplicidad funcional y la integridad de los datos bajo una arquitectura clásica y robusta.

## 🛠️ Especificaciones Técnicas
* **Arquitectura:** Patrón Modelo-Vista-Controlador (MVC) + DAO (Data Access Object).
* **Backend:** Java (Servlets).
* **Frontend:** JSP, HTML5, CSS3.
* **Base de Datos:** MySQL.
* **Gestión de Dependencias:** Maven.

## 📋 Requisitos del Sistema
El sistema cumple con las siguientes funcionalidades clave:
* **Autenticación:** Sistema de login para usuarios autorizados.
* **CRUD de Productos:** Gestión completa (Crear, Leer, Actualizar, Eliminar).
* **Movimientos de Inventario:** Registro de entradas y salidas de mercancía.
* **Actualización Automática:** El stock se sincroniza en tiempo real con cada movimiento.
* **Alertas Visuales:** Notificaciones de stock mínimo para evitar desabastecimiento.

## 🏗️ Estructura del Proyecto (POJOs)
El modelo de datos incluye las entidades principales:
- `Usuario`: Gestión de acceso.
- `Producto`: Información detallada de los artículos.
- `Movimiento`: Histórico de transacciones de inventario.
